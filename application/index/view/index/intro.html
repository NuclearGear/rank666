功能介绍

  • 差价统计
  • 单品查询
  • 购买统计
  • 主页中心
  1. 差价统计——自动计算出 stockx 和 毒App鞋 差价大于100的鞋子
  2. 近期波动：点击名称可查看近期 毒App 价格、销量折线图。
  3. 便捷搜索：可根据 名称、货号搜索。尺码、发售、差价、销量、利率排序。
  4. 一键直达：有stockx网站鞋款直达链接，一键直达。立即购买。
  1. 单品查询——根据需要，对毒APP的单品进行搜索。
  2. 尺码销量：毒APP平台的鞋款各个尺码的销量情况展示。
  3. 价格折线图：所搜索的鞋款近期毒APP的价格涨幅走势情况。
  4. 销量折线图：所搜索的鞋款近期毒APP的销量涨幅走势情况。
  1. 购买统计——添加已经购买的商品，搜索已经购买的商品。
  2. 利润统计：有实际利润(成本价与实际卖出的价格相比所得的利润)和预计利润(成本价与毒APP的价格相比得出的利润)。
  3. 利率比例：可以非常直观的反应出每月的盈亏情况。
  4. 鞋子状态：待收货、待出售、已出售，全部直接展示在下方，简洁直观。
  5. 市场波动：点击名称可查看近期 毒App 价格、销量折线图。
  1. 主页中心——根据用户本月已购买的鞋款进行统计。
  2. 销售情况：本月盈利、成本、销售额、利率比与上月进行比较涨跌幅直观展示，非常直观的展示本月与上月的销售情况对比。
  3. 相关鞋款：本月购买相同鞋款数量最多的前四款鞋子的近期价格走势折线图。
  4. 转运信息：根据用户选择的转运平台，转运信息会直接显示在下方，不需要在去转运官网查询，方便快捷。(注：此功能正在开发中)
